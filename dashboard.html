Sistem AI Menunggu Konfigurasi

Isi minimal Slot 1 (Key Utama) di menu samping kiri, klik Simpan & Hubungkan, lalu pilih modul aplikasi.

Cara Mendapatkan API Key

  1. Kunjungi situs resmi Google AI Studio.
  2. Login menggunakan Akun Google Anda.
  3. Klik "Get API key" lalu "Create API key".
  4. Salin kode kunci dan tempelkan ke Slot 1 di Dashboard ini.
