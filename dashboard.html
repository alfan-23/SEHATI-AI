Sistem AI Menunggu Konfigurasi

Buka Menu Bar, isi minimal Slot 1 (Key Utama), klik Simpan & Hubungkan, lalu pilih modul aplikasi.

Cara Mendapatkan API Key

  1. Kunjungi situs resmi Google AI Studio.
  2. Login menggunakan Akun Google Anda.
  3. Klik tombol biru bertuliskan "Get API key" di menu sebelah kiri.
  4. Klik tombol "Create API key", lalu salin (copy) kode kunci yang muncul.
  5. Tempelkan (paste) kunci tersebut ke Slot 1 di Dashboard ini, lalu klik Simpan & Hubungkan.
